What is Call Masking Fraud?

Understanding the threat we protect against

Call Masking Explained

Definition: Call masking (CLI spoofing) is when fraudsters use multiple fake caller IDs to reach the same victim.

How Attackers Operate:

  • Generate dozens of fake phone numbers
  • Call the same victim from different "identities"
  • Evade traditional fraud detection
  • Execute scams with apparent legitimacy

The Attack Pattern

    Attacker System                              Victim
    ┌─────────────┐                         ┌─────────────┐
    │  Generates  │                         │             │
    │   Fake IDs  │                         │  +234-XXX   │
    └──────┬──────┘                         └──────▲──────┘

How Our System Works

Real-time fraud detection in milliseconds

Detection Logic

We Monitor:

  • A-Numbers: Callers (source)
  • B-Numbers: Destinations (target)
  • Time Windows: Calls within seconds

We Detect:

  • 5+ different callers
  • Same destination
  • Within 5 seconds
Result: Fraudulent calls are detected and can be automatically disconnected.

NCC Compliance

Regulatory Framework

The Nigerian Communications Commission (NCC) requires all telecom operators to implement fraud detection and report incidents.

  • 5-year audit log retention
